# GuardRail Compliance Engine

GuardRail Compliance Engine parses infrastructure definitions, normalizes resources into deterministic facts, evaluates them against compliance policies, and emits console, JSON, SARIF, or HTML output.

Current implementation coverage includes Terraform, CloudFormation, and Kubernetes parsing with AWS-focused starter rule packs.

See `getting-started.md` for clone/install steps, AWS CLI setup, live Bedrock configuration, and example run commands.
